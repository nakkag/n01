# コンピュータ対戦用アレンジ
#
# 残り点数によって狙う場所を決めます。
# 同じ残り点数で複数個ある場合はランダム(確立により変動)に選択されます。
#
# [形式]
# 残り点数,1本目の場合,2本目の場合,3本目の場合,繰り返し数,相手の上がり目フラグ(0-両方 1-上がり目あり 2-上がり目なし)
#
# 繰り返し数の指定が無い場合は、複数個ある場合に同じ確立で選択されます。
# たとえば、
#
# 10,D5,D5,D5,3
# 10,S2,D5,D5,2
#
# は以下のように展開されます。 
#
# 10,D5,D5,D5
# 10,D5,D5,D5
# 10,D5,D5,D5
# 10,S2,D5,D5
# 10,S2,D5,D5
#
# 上がり目フラグは、相手が上がり目が出ているかどうかを判定するフラグです。
# 上がり目フラグの指定が無い場合は、0 を指定したのと同じになります。
#
